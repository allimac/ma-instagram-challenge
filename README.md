## How to use it
Two ways to check this project out: either at .... or following the instructions below to run it on your machine:
```
$ git clone git@github.com:allimac/ma-instagram-challenge.git
$ bundle install
$ rake db:create
$ rake db:migrate
```
- Start the server
```
$ rails s
```
- Visit localhost:3000
- Check out the tests
```
$ rspec
```
